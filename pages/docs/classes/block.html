Создание блока

Block

Является главным классом, который содержит основую логику работы блоков.
Наследуется: EnergyBlock

Методы

get_meta(), get_all_meta(), set_meta(), remove_meta(), get_position(), get_id(), get_type(), is_missing(), tick(), run_logic(), get_logic()

get_meta()

Данный метод принимает 1 агрумент (key), который является ключем для значения. Возвращяет хранимые данные по данному ключу.

get_all_meta()

Данный метод возвращяет таблицу со всеми элементами меты в блоке.

set_meta()

Данный метод принимает 2 агрумента (key, value), где 1 агрумент это ключ, а второй агрумент значение. Создает или изменяет уже существующие данные(мету) у блока по ключу.

remove_meta()

Данный метод принимает 1 агрумент (key), который является ключем для значения. Удаляет хранимые данные по данному ключу.

get_position()

Данный метод возвраящет позицию блока (3 значения подряд).

get_id()

Возвращяет id блока в формате mod_id:block_id

get_type()

Возвращяет тип блока. Тип блока является строкой из Enum BlockType

is_missing()

Возвращяет true, если блок не прогружен (находится за дистанцией прорисовки)

tick()

Данный метод вызывается автоматический каждый тик, так же вызывает run_logic()

run_logic()

Вызывается автоматический каждый тик если есть функция. Задать функцию можно с помощью метода CreateBlockFunc()

get_logic()

Возвращяет функцию данного блока
