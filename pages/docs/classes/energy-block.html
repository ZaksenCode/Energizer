Создание блока

EnergyBlock

Является основным классом для блоков с энергией, который содержит логику работы с энергией.
Наследует: Block
Наследуется: Machine, Wire

Методы

get_network(), set_network(), get_energy(), get_max_energy(), set_energy(), is_energy_max(), count_max_energy(), receive_energy(), try_use_energy(), give_energy(), give_energy_neighbours()

get_network()

Возвраящет индекс сети в которой он находится

set_network()

Принимает один аргумент (index), который является индексом сети на которую нужно сменить блок.
P.S. не рекомендуется использовать на прямую

get_energy()

Возвращяет текущее количество энергии в блоке

get_max_energy()

Возвращяет максимальное количество энергии в блоке

set_energy()

Задает количетсво энергии в блоке.
Принимает один агрумент (count), количество энергии.

is_energy_max()

Возвращяет true если блок полностью заполнен энергие

count_max_energy()

Принимает 1 агрумент (count). Возвращяет количество энергии, которое может вместить из агрумента.

receive_energy()

Вызывается когда блок получает энергию.

try_use_energy()

Принимает 1 агрумент (count), количство энергии, которое попробует использовать. Вернет true если ему он сможет её потратить и потратит её.

give_energy()

Отдаёт свою энергию другому блоку. Принимает 2 агрумента (energy_block, count), первый агрумент объект класса, второй агрумент количство энергии.

give_energy_neighbours()

Отдаёт свою энергию всем соседним блокам, которые могу её получить. Принимает 1 агрумент (count). Делит энергию равномерно между всеми доступными блоками.
