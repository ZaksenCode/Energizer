Logger

Является небольшим утилитарным классом для логирования.

Методы

get_name(), log(), err(), warn(), debug(), table()

get_name()

Возвращяет имя логгера в формате [name]

log()

Выводит сообщение в консоль

err()

Выводит ошибку в консоль. Сообщение с припиской error:

warn()

Выводит предупреждение в консоль. Сообщение с припиской warn:

debug()

Выводит сообщение отладки в консоль. Сообщение с припиской debug:. Работает только в случае, когда логгеру задан debug режим.

table()

Выводит таблицу в консоль. Работает только в случае, когда логгеру задан debug режим.

Как создать logger

Для создания логгера вам необходимо прописать следующее:
Logger = Logger:new("mod_id", is_debug)
Это создаст Logger с вашим mod_id, вторым агрумент нужно указать, работает ли ваш логгер с отладкой (true/false).
