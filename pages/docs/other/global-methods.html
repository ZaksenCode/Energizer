Глобальные методы

Набо методов которые вам пригодятся для работы с Energizer.

Методы

CreateBlock(), CreateMachine(), CreateWire(), GetBlock(), RemoveBlock(), GetBlockNeighbours(), AttachToNetwork(), DetachFromNetwork(), GetMachinesIn(), GetRecipientsIn(), CreateBlockType(), CreateBlockFunc()

CreateBlock()

Создает объект класса Block и возвращяет его. Принимает 6 агрументов (x, y, z, id, mod_id, meta).
x, y, z - позиция блока.
id - ид блока.
mod_id - ид мода
meta - таблица с данными
Пример: Block

CreateMachine()

Создает объект класса Machine и возвращяет его. Принимает 6 агрументов (x, y, z, id, mod_id, meta).
x, y, z - позиция блока.
id - ид блока.
mod_id - ид мода
meta - таблица с данными
Пример: Machine

CreateWire()

Создает объект класса Wire и возвращяет его. Принимает 6 агрументов (x, y, z, id, mod_id, meta).
x, y, z - позиция блока.
id - ид блока.
mod_id - ид мода
meta - таблица с данными
Пример: Wire

GetBlock()

Вовращяет объект класса на текущий позиции, принимает 3 агрумента (x, y, z)

RemoveBlock()

Удаляет объект класса на текущий позиции, принимает 3 агрумента (x, y, z)

GetBlockNeighbours()

Получает все соседнии объекты классов

AttachToNetwork()

Добавляет блок в сеть

DetachFromNetwork()

Удаляет блок их сети

GetMachinesIn()

Получает все машины в сети

GetRecipientsIn()

Получает все машины типа получатель в сети

CreateBlockType()

Создает новый тип блока, принимает 3 агрумента (type, value, func). Первый агрумент это тип блока, например Machine. Строковое значение типа блока, например machine. Функция загрузки блока их файла, например:
function (x, y, z, id, mod_id, meta)
CreateMachine(x, y, z, id, mod_id, meta)
end

CreateBlockFunc()

Создает функцию для блока, принимает 2 агрумента (block_id, func).
block_id - ид блока для которого будет создана функция.
func - функция принимающая 1 агрумент (block)
block - объект класса
Например:
function (machine)
print("tick!")
end
Пример: Block
