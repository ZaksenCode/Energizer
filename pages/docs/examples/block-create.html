Создание блока

Показывает примеры правильного создания блоков и описывает их.

Блоки

Block, Machine, Wire

Block

Для создание блока вам необходимо просать в его скрипте:
-- Данная функция вызывается при установки блока
function on_placed(x, y, z)
CreateBlock(x, y, z, "block_id", "mod_id", {})
end
-- Данная функция вызывается при установки блока
function on_broken(x, y, z)
RemoveBlock(x, y, z)
end
Если вы хотите создать функцию для блока, вам необходимо вызвать функцию CreateBlockFunc() из world.lua скрипта:
CreateBlockFunc("machine", function (machine)
print("tick!")
end)

Machine

Для создание механизма вам необходимо просать в его скрипте:
-- Данная функция вызывается при установки блока
function on_placed(x, y, z)
CreateBlock(x, y, z, "block_id", "mod_id", {})
AttachToNetwork(x, y, z)
end
-- Данная функция вызывается при установки блока
function on_broken(x, y, z)
DetachFromNetwork(x, y, z)
RemoveBlock(x, y, z)
end
